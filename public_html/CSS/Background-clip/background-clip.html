The background-clip CSS property specifies whether an element's background, either the color or image, extends underneath its border.

If no background image or color is set, this property will only have a visual effect when the border has transparent regions or partially opaque regions (due to border-style or border-image); otherwise the border covers up the difference.

Syntax border-box|padding-box|content-box|initial|inherit
JavaScript syntax object.style.backgroundClip="content-box"
Initial value border-box
Applies to
Inherited
Percentages
Computed value
Animatable
Version: CSS3

  • border-box

    Default value. The background is clipped to the border box

  • padding-box

    The background is clipped to the padding box

  • content-box

    The background is clipped to the content box
