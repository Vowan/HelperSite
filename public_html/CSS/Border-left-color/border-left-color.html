The border-left-color CSS property sets the color of the left border of an element. Note that in many cases the shorthand CSS properties border-color or border-left are more convenient and preferable.

Syntax

border-left-color: red;
border-left-color: rgb(255, 255, 0);
border-left-color: hsla(100%, 50%, 25%, 0.75);
border-left-color: transparent;
border-left-color: #ffbb00;
border-left-color: currentColor;
border-left-color: inherit;
Initial value currentColor
Applies to all elements. It also applies to ::first-letter.
Inherited no
Percentages -
Media visual
Computed value If the value is translucent, the computed value will be the rgba() corresponding one. If it isn't, it will be the rgb() corresponding one. The transparent keyword maps to rgba(0,0,0,0).
Animatable yes, as a color
Canonical order the unique non-ambiguous order defined by the formal grammar

Values

  • Is a <color> CSS value describing the color of the left border.

  • inherit

    Is a keyword denoting the color of the left border of the parent's element (which may be different from the border-left-color default value)
