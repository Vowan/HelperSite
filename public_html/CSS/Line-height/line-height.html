The line-height property specifies the line height.

Syntax normal|number|length|initial|inherit
JavaScript syntax object.style.lineHeight="30px"
Initial value
Applies to
Inherited
Percentages
Computed value
Animatable
Version:

  • normal

    A normal line height. This is default

  • number

    A number that will be multiplied with the current font size to set the line height

  • length

    A fixed line height in px, pt, cm, etc.

  • %

    A line height in percent of the current font size
