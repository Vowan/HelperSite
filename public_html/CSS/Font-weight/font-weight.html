The font-weight CSS property specifies the weight or boldness of the font. Some fonts are only available in normal and bold. 

Syntax

font-weight: normal;
font-weight: bold;

/* Relative to the parent */
font-weight: lighter;
font-weight: bolder;

font-weight: 100;
font-weight: 200;
font-weight: 300;
font-weight: 400;
font-weight: 500;
font-weight: 600;
font-weight: 700;
font-weight: 800;
font-weight: 900;

/* Global values */
font-weight: inherit;
font-weight: initial;
font-weight: unset;

Version:

  • normal

    Normal font weight. Same as 400.

  • bold

    Bold font weight. Same as 700.

  • lighter

    One font weight lighter than the parent element (among the available weights of the font).

  • bolder

    One font weight darker than the parent element (among the available weights of the font).

  • 100, 200, 300, 400, 500, 600, 700, 800, 900

    Numeric font weights for fonts that provide more than just normal and bold.
