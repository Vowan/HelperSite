The background-color CSS property sets the background color of an element, either through a color value or the keyword transparent.

Syntax color|transparent|initial|inherit
JavaScript syntax object.style.backgroundColor="#00FF00"
Initial value transparent
Applies to
Inherited
Percentages
Computed value If the value is translucent, the computed value will be the rgba() corresponding one. If it isn't, it will be the rgb() corresponding one. The transparent keyword maps to rgba(0,0,0,0).
Animatable yes, as a color
Version: CSS1

  • Specifies the background color. Look at CSS Color Values for a complete list of possible color values.

  • transparent

    Specifies that the background color should be transparent. This is default
