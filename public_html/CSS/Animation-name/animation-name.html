The animation-name CSS property specifies a list of animations that should be applied to the selected element. Each name indicates a @keyframes at-rule that defines the property values for the animation sequence.

Syntax keyframename|none|initial|inherit
JavaScript syntax object.style.animationName="myNEWmove"
Initial value none
Applies to
Inherited
Percentages
Computed value
Animatable
Version: CSS3

  • keyframename

    Specifies the name of the keyframe you want to bind to the selector

  • none

    Default value. Specifies that there will be no animation (can be used to override animations coming from the cascade)
