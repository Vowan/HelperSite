The animation-direction property defines whether an animation should play in reverse direction or in alternate cycles.

Syntax normal|reverse|alternate|alternate-reverse|initial|inherit
JavaScript syntax object.style.animationDirection="reverse"
Initial value normal
Applies to
Inherited
Percentages
Computed value
Animatable
Version: CSS3

  • normal

    Default value. The animation should be played as normal

  • reverse

    The animation should play in reverse direction

  • alternate

    The animation will be played as normal every odd time (1,3,5,etc..) and in reverse direction every even time (2,4,6,etc...)

  • alternate-reverse

    The animation will be played in reverse direction every odd time (1,3,5,etc..) and in a normal direction every even time (2,4,6,etc...)
