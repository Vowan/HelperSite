The border-top-width CSS property sets the width of the top border of a box.

Formal syntax

How to read CSS syntax.

<br-width> where <br-width> = <length> | thin | medium | thick

Initial value medium
Applies to all elements. It also applies to ::first-letter.
Inherited no
Percentages -
Media visual
Computed value the absolute length or 0 if border-top-style is none or hidden
Animatable yes, as a length
Canonical order the unique non-ambiguous order defined by the formal grammar

  • <br-width>

    Is either a non-negative explicit value or a keyword denoting the thickness of the bottom border. The keyword must be one of the following values:

    • thin

    • medium

    • thick

  • inherit

    Represents the calculated value of the border-top-width of the parent's element.
