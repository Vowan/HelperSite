The animation-iteration-count CSS property defines the number of times an animation cycle should be played before stopping.

Syntax number|infinite|initial|inherit
JavaScript syntax object.style.animationIterationCount="infinite"
Initial value 1
Applies to
Inherited
Percentages
Computed value
Animatable
Version: CSS3

  • number

    A number that defines how many times an animation should be played. Default value is 1

  • infinite

    Specifies that the animation should be played infinite times (for ever)
